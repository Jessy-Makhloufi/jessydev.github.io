À propos de moi

Bonjour ! Je m'appelle Jessy et je suis passionné par la programmation et les jeux vidéo. Mon objectif est de faire de cette passion mon futur métier.
Les projets ci-dessous sont codés en Python. Il est donc indispensable d'installer le langage de programmation Python en version 3.12.10 pour faire fonctionner ces projets. Voici le lien pour télécharger Python : Installer Python 3.12.10

Mes Projets

Site Web – Jeux vidéo 2023

Site web réalisé en groupe en HTML et CSS présentant l’histoire du jeu vidéo, les types de jeux, les entreprises du secteur, les composants d’un PC gaming et les créateurs de contenu les plus influents.


Langage de programmation : HTML, CSS, Javascript

Horloge Interactive

Application développée en Python permettant d’afficher l’heure en temps réel. L’utilisateur peut ajouter ou retirer des heures et des minutes, et modifier le format d’affichage.


Langage de programmation : Python

Convertisseur Binaire

Application Python permettant de convertir des nombres décimaux en binaire, en hexadécimal et inversement.


Langage de programmation : Python

Éditeur Pixel Art

Application développée en Python permettant de modifier et personnaliser des pixel arts prédéfinis.


Langage de programmation : Python

Mini-jeu de Tri

Mini-jeu développé en Python dans lequel le joueur doit trier des billes selon leur couleur.


Langage de programmation : Python

Projet – Nuit du Code

Jeu développé dans le cadre de l’événement « La Nuit du Code », dont l’objectif est de créer un jeu en Python en 6 heures. Version améliorée de mon exercice d’entraînement. (Pour faire fonctionner le jeu, il faut installer le module Pixel pour Python)
